# Color-Converter
[Chorme Extension] Color Converter : RGB to Hex, Hex to RGB

## 개요
- Hex -> RGB, RGB -> HEX 작업 빈도가 증가하여 크롬확장프로그램으로 두고 사용하면 편리할 것 같아서 개발 진행
- 0,0,0 쉼표로 구분된 세 숫자를 입력하면 RGB코드로 인식하여 HEX값을 반환
- #FFFFFF 와 같이 입력하면 HEX코드로 인식하여 RGB값으로 반환
